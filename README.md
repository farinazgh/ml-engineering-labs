# Cloud to ML Engineer

A hands-on learning journey from Cloud & DevOps Architecture to Machine Learning Engineering.

This repository documents my progress through Python, Machine Learning, Deep Learning, and production-oriented AI engineering projects.

The goal is not only to learn ML theory, but also to approach AI systems with an engineering and cloud-native mindset:
- clean project structure
- reproducible experiments
- scalable architectures
- deployment workflows
- MLOps practices
- real-world projects
